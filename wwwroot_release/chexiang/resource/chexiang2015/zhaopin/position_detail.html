  • 发布时间:2015-04-18
  • 工作地点:上海市
  • 招聘人数:1
  • 学历要求:本科及以上
  • 工作性质:全职
申请职位

岗位职责

  1. 建立健全公司财务管理、预算管理、会计核算、稽核审计等有关制度，监督各项制度的实施和执行；
  2. 参与制订公司年度总预算和半年度预算调整，汇总、审核下级部门上报的年度预算，召集并主持公司年度预算分析与平衡会议；
  3. 掌握公司财务状况、经营成果和资金变动情况，及时向董事会汇报工作情况；
  4. 负责审核签署公司预算、财务收支计划、成本费用计划、财务报告、会计决算报表，会签涉及财务收支的重大业务计划、经济合同、经济协议等；

任职资格

  1. 财务会计专业或金融专业本科以上学历，有注册会计师资格，年龄30-40周岁,有四大工作背景者优先；
  2. 具有企业财务领导岗位五年以上工作经验，有上市公司财务经理以上职务一年以上经验者优先；
  3. 掌握公司财务状况、经营成果和资金变动情况，及时向董事会汇报工作情况；
  4. 熟悉国家相关财务、税务、审计法规及政策，精通财务、金融专业知识；
